# DOM Manipulation Practice

This repository contains beginner-friendly JavaScript DOM manipulation exercises.
Each exercise is organized in its own folder with separate HTML, CSS, and JavaScript files.

## Exercises

1. `dark-mode-toggle` - Toggle between dark mode and light mode.
2. `image-generator` - Generate images with a loading spinner.
3. `image-switch` - Click an image to switch to the next one.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Run Locally

1. Clone or download this repository.
2. Open the folder in VS Code.
3. Open any exercise folder.
4. Run `index.html` in a browser (or use Live Server).
